# Uranium

Current version is 1.0.1-pre

Transfer Files or Directories Between Linux and Windows Without SSH Support

## Requirements
### Running
- linux(ubuntu)  linux_release
    make clean; make clean-cache;
    make -j8; make linux_release
- windows        release
    make clean; make clean-cache
    make -j8
### Building
- cygwin
- linux(ubuntu)
